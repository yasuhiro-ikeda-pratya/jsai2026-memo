## LLMに組み込まれている規範と、瞑想が目指す規範
- LLMには「親切・無害・協力的」という規範が注入されている  
- 協力はどう安定するか \- 間接互恵性と評判  
  - 一次の規範：「何をしたか」だけで評判が決まる  
  - 二次の規範：「誰に対してしたか」も考慮する  
- LLM集団の協力はフリーライダーの侵入に脆い  
  - LLMは一次規範に留まり、二次規範に弱い  
- 「ゴシップ」分散型の評判が協力を安定させる  
- 瞑想における社会的規範  
  - 規範を外から注入して行動を修正するのではなく、すでに備わった自然な状態を「認識する」→LLMの規範形成と正反対の方向性  
- 戒・律は理想の状態に至るための「後付けの足場」  
  - 「外的規範は体験的理解を経て不要になる足場」と言う仏教の規範観をLLMの学習・アラインメントに応用できないか
